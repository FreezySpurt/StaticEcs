# Roslyn analyzer

StaticEcs has a dedicated Roslyn analyzer and code-fix suite that catches common framework misuses at compile time. It is shipped as a separate package — `FFS.StaticEcs.Analyzers` on NuGet and `com.felid-force-studios.static-ecs-analyzer` on Unity UPM.

👉 **Full documentation, installation instructions and the complete rule list:** [github.com/Felid-Force-Studios/StaticEcs-Analyzer](https://github.com/Felid-Force-Studios/StaticEcs-Analyzer)
