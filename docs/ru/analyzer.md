# Roslyn-анализатор

У StaticEcs есть отдельный набор Roslyn-анализаторов и code-fix'ов, который ловит типовые ошибки использования фреймворка во время компиляции. Поставляется отдельным пакетом — `FFS.StaticEcs.Analyzers` на NuGet и `com.felid-force-studios.static-ecs-analyzer` в Unity UPM.

👉 **Полная документация, инструкции по установке и список правил:** [github.com/Felid-Force-Studios/StaticEcs-Analyzer](https://github.com/Felid-Force-Studios/StaticEcs-Analyzer)
