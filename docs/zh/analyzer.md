# Roslyn 分析器

StaticEcs 提供一套独立的 Roslyn 分析器与代码修复 (code-fix)，在编译期捕获框架的常见误用。以独立包形式发布 —— NuGet 上的 `FFS.StaticEcs.Analyzers` 和 Unity UPM 上的 `com.felid-force-studios.static-ecs-analyzer`。

👉 **完整文档、安装说明与规则列表：** [github.com/Felid-Force-Studios/StaticEcs-Analyzer](https://github.com/Felid-Force-Studios/StaticEcs-Analyzer)
